A KING'S PROMISE — FAVICON DEPLOYMENT PACKAGE
============================================

Drop every file in this folder into the ROOT of your website
(the same folder that holds index.html).

Then paste this into the <head> of every page:

<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#000000">

Squarespace / Wix / WordPress / Shopify: upload favicon.ico in the
platform's "favicon" or "site icon" setting (or icon-512.png if PNG required).

After upload, hard-refresh (Cmd-Shift-R / Ctrl-Shift-F5) or open in
incognito to verify; favicons cache aggressively.
